/* 该软件是本人在提升QT开发过程中的练手项目 ,仅供学习*/  2020-10-6

工程介绍：
android: 将项目打包成安卓APK时，创建了一些相应的配置信息；
CustomControl: 该文件夹内包含了，项目中常用的自定义控件，非常精美，有些是借鉴其他大神的作品。具有较强的移植性
image: 里面存放项目需要的图标等资源文件

初学者或者接触QT两年内都可以参考该项目。

本项目是监护仪软件，通常跑在Linux上。而且监护仪软件的功能十分强大，本项目也只是把监护仪的功能实现了冰山一角。
为甚么要写这个项目，因为项目中包含了许多QT的重要元素，可以在写这个软件的过程中提高自己的编程能力和对QT开发的了解
本项目包含的知识点大致如下：后期不定期新增或修改

1.控件自适应：美观，在将项目移植到安卓下的时候，界面是PC端一样整齐美观，如果不做自适应，会出现 界面显示不全，布局被打乱的现象
2.QT中使用样式表，全局设置控件样式。
3.QT中的绘图事件，深入熟悉QT的绘图机制
4.自己模拟数值，绘制相应的波形
5.QT中各个控件的熟悉，以及使用场景
6.一些较复杂的自定义控件的绘制
7.在qt中使用MVC架构
